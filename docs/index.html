How to Get a GitHub Token

  1. Open https://github.com/settings/tokens/new
  2. If prompted, choose Generate new token (classic)
  3. Note: enter a name like "M2B Data Access"
  4. Expiration: choose a date (or "No expiration" if you prefer)
  5. Scopes: check repo (full control of private repositories)
  6. Scroll down and click Generate token
  7. Copy the token immediately (starts with ghp_)
  8. Paste it into the GitHub Token field and click Save Settings

Important: GitHub only shows the token once. Save it in a password manager.
